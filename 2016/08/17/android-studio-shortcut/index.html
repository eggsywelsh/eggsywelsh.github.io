android studio 快捷键(本文持续更新)

发表于 2016-08-17   |   分类于 android   |     |   阅读次数

介绍android studio的常用快捷键

前提：android studio版本Android Studio 2.2 Preview 6

原来一直使用Eclipse开发android，google官方早在2013年5月就推出了android studio，苦于一直在赶项目，没有时间切换过来使用，前几个月终于有时间了，把开发环境切到studio，发现太好用了，但是刚开始用的时候，快捷键和Eclipse相差太大，虽然可以使用Eclipse快捷键，但是吧，既然使用了，就要原汁原味的用好它，现在下面总结一下studio下面的快捷键：

功能 Android studio
查找文件 Ctrl+Shift+N
查找类 Ctrl+N
查找类中的方法或变量 Ctrl+Shift+Alt+N
导入包 Alt+回车
显示当前文件的结构 待定
格式化代码 Ctrl+Alt+L(与Ubuntu中锁屏键冲突)
管理导入的类 Ctrl+Alt+O
当前行的内容往上或下移动 Ctrl+Shift+方向键
重构代码 Ctrl+Shift+Alt+t
为JavaBean快速生成getter/setter、toString()、以及构造方法等 Alt+Insert
重命名 Shift+F6
格式化代码 Ctrl+Alt+L
代码助手 Ctrl+Shift+Space 自动补全代码
Ctrl+空格 代码提示(会与输入法切换有冲突)
Ctrl+Alt+Space 类名或接口名提示
Ctrl+P 方法参数提示
行操作 Ctrl+X 删除行
Ctrl+D 复制行
Alt+Shift+Up/Down 代码向上/下移动
返回至上次浏览的位置 Ctrl+Alt+left/right(与ubuntu切换视图冲突)
在方法间快速移动定位 Alt+Up/Down
切换代码视图 Alt+left/right
注释 Ctrl+/ 或 Ctrl+Shift+/ （// 或者/…/ ）
最近更改的代码 Alt+Shift+C
重命名 Shift+F6
删除并剪贴行 Ctrl+X
复制一行 Ctrl+D
打开最近编辑的文件列表 Ctrl+E
当前页面替换文本 Ctrl+R
全局替换文本 Ctrl+Shift+R
当前页面查找文本 Ctrl+F
全局页面查找文本 Ctrl+Shift+F
删除行 Ctrl+Y
重写或实现父类 Ctrl+O
自动代码 Ctrl+J
显示类继承结构图 Ctrl+H
显示注释文档 Ctrl+Q
撤销 Ctrl+Z
恢复撤销 Ctrl+Shift+Z
重构 Ctrl+Alt+Shift+T
大小写转换 Ctrl+Shif+U
当前页面快速展开代码 Ctrl+Shift+”+“
当前页面快速折叠代码 Ctrl+Shift+”-“
光标所在当前方法展开代码 Ctrl+”+“
光标所在当前方法折叠代码 Ctrl+”-“
快捷定位到行首/尾 Ctrl + Left/Right
当前方法的声明 Alt+Q
快速打开光标处的类或方法 Ctrl+B
可以把代码包在一块内 Ctrl＋Alt＋T，例如try/catch(与ubuntu快捷键冲突)
在不离开当前类的前提下查看一个方法的实现 Ctrl+shift+i
高亮错误或警告快速定位 F2或Shift+F2
高亮显示所有该文体，按Esc高亮消失 Ctrl+Shift+F7
鼠标左键+Alt 列选择模式
快速定位到当前类报错或者有警告的地方 F2或者Shift+F2

持续更新中………
